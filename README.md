# Lista de Exercícios: Lógica de Programação em C 🚀

Este repositório contém as soluções para uma lista de exercícios focada no desenvolvimento de lógica e estruturas de controle (laços de repetição e condicionais) na linguagem C.

## 📋 Exercícios Implementados

O projeto aborda os seguintes desafios de programação:

1.  **Intervalo de Inteiros**: Exibe todos os números entre dois valores informados.
2.  **Análise de Valores**: Identifica o maior e o menor entre 10 números reais.
3.  **Cálculo de Série**: Processamento da soma fracionária: $S = \frac{1}{1} + \frac{3}{2} + \dots + \frac{99}{50}$.
4.  **Produto Indeterminado**: Calcula o produto de N números até que o valor zero seja digitado.
5.  **Filtro Numérico**: Gera números entre 1000 e 1999 com resto 5 na divisão por 11.
6.  **Sequência de Fibonacci**: Gera os $n$ primeiros termos da sequência clássica.
7.  **Soma por Paridade**: Acumula valores pares e ímpares até encontrar um número > 1000.
8.  **Números Primos**: Identifica e exibe os $n$ primeiros números primos.
9.  **Soma de Primos em Intervalo**: Calcula o somatório de primos entre os pontos A e B.
10. **Inversão de Inteiros**: Algoritmo para inverter a ordem dos dígitos de um número (Ex: 1234 -> 4321).
11. **Tabuadas Dinâmicas**: Exibe as tabuadas de 1 a 10 utilizando laços aninhados.
12. **Estatística de Dados**: Calcula a média e a contagem de valores até a entrada de um sinal de parada (zero).

## 🛠️ Tecnologias Utilizadas

*   **Linguagem C** (Padrão C11/C17)
*   **GCC** (GNU Compiler Collection)
